# Biblioteca WEB API REST (ASP.NET)

Tecnologias utilizadas: ASP.NET Core Web API • C# • Entity Framework Core • PostgreSQL • JWT Authentication • AutoMapper • Docker • xUnit • Mock

Projeto desenvolvido para aprimoramento pessoal e aperfeiçoamento prático no desenvolvimento de APIs REST utilizando ASP.NET Core. O principal objetivo foi aplicar conceitos e práticas utilizadas no mercado, buscando construir uma aplicação organizada, escalável e de fácil manutenção.

A aplicação foi desenvolvida utilizando os princípios de Clean Architecture, com foco em separação de responsabilidades, autenticação JWT, boas práticas de estruturação de código, tratamento global de exceções, testes automatizados e organização em camadas.

O sistema permite o gerenciamento de livros, categorias, usuários e solicitações de empréstimo, utilizando PostgreSQL com Entity Framework Core para persistência de dados e Swagger para documentação da API.

Funcionalidades
Autenticação com JWT
Cadastro e gerenciamento de livros
Cadastro de categorias
Solicitações de empréstimo
Paginação de resultados
Tratamento global de exceções
Rate Limiting
Testes unitários e de integração
